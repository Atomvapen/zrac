## Design

* Exlusively written in [Zig].
* Use as little dependencies as possible.
* Compliant with, and based upon [SÄKR EHV/PV].
* Foremost be a good _learning experience_ - not a good _product_.

## Dependencies

* [zgui-rl] 
* [raylib-zig]
